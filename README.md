# Mini-Exp

Mini-Exp is a collection of my mini curious experiments.

The goal is to keep each experiment self-contained in its own folder so anyone can follow along, rerun the work, inspect the artifacts, or challenge the results.

## Experiments

- [cot-faithfulness](/Volumes/Projects/Mini%20experiments/cot-faithfulness): a small study on whether visible chain-of-thought in open-source LLMs actually supports the final answer they give.
